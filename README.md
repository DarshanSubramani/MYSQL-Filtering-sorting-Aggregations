Sales Revenue SQL Assignment – Task 3

 Dataset
The dataset `salesrevenue` contains sales information with the following columns:

CustomerID` – Unique ID for each customer
Name` – Customer name
Age` – Customer age
Gender` – Customer gender
State` – Customer state
Region` – Region of the customer
ProductCategory` – Category of product purchased
Revenue` – Revenue generated
Month` – Month of purchase

Total records: 10

 SQL Queries
1. View all records (`SELECT *`)  
2. Count total records (`COUNT(*)`)  
3. Filter data using `WHERE` (by region or revenue)  
4. Sort data using `ORDER BY`)  
5. Aggregate revenue using `GROUP BY`)  
6. Filter aggregated results using `HAVING`)  

All queries are included in `queries_task3.sql'
